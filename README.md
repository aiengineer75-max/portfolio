# Personal Portfolio Website

## Overview

This is my personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio showcases my skills, projects, education, and contact information in a modern and responsive design.

The purpose of this project is to present my work, technical skills, and academic background in a professional way.

---

## Features

* Responsive Navigation Bar
* Mobile Menu
* Hero Section with Typewriter Effect
* About Me Section
* Skills Section with Progress Bars
* Projects Showcase
* Services Section
* Education Timeline
* Contact Information
* Scroll to Top Button
* Loading Screen Animation
* AOS (Animate On Scroll) Animations
* Responsive Design for Different Screen Sizes

---

## Technologies Used

### Frontend
